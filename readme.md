# Build

At this time, the source is build with VisualStudio 2015.

##build from Command Prompt.

>MSBuild CGLib.sln

##Build from GUI.

Open CGLib.sln

[menu] -> [Build] -> [Build Solution] 

I will build other platforms( mac, linux ).
